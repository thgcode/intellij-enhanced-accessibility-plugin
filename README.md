# IntelliJ Enhanced Accessibility Plugin

## Warning

This plugin is pre-alpha.

## What is it?

This plugin provides enhanced accessibility functionality for the IntelliJ IDE
for Screen Reader users.

## Features

* Plays and speaks errors and warnings whenever the cursor moves to a new line.
To use it, just install the plugin. Whenever the cursor moves to a new line,
IntelliJ will play a sound based on the type (error or warning) and NVDA will speak the error description.
